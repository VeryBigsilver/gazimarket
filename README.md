# 가지마켓(gazimarket)

DESIGN.md 요구사항에 맞춰 만든 Flask + SQLite 기반 중고거래 플랫폼입니다.

## 문서

- [설치 및 실행 방법](SETUP.md)
- [요구사항 및 시스템 설계](DESIGN.md)

## 포함 기능

- 회원가입, 로그인, 마이페이지
- 상품 등록, 검색, 상세 조회, 수정, 삭제
- 가상 포인트 송금 및 거래 완료 처리
- 전체 채팅 및 상품별 1:1 채팅
- 상품/사용자 신고
- 관리자 회원, 상품, 신고 관리

## 기본 관리자 계정
